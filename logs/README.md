# Logs
### *FlowX – Test Rig Telemetry Logs*

The `logs/` directory stores CSV telemetry files generated by `hybrid_rate_controller.py` during test rig sessions. Each file is timestamped at the moment the controller is launched.

---

## File Naming

```
rig_log_YYYYMMDD_HHMMSS.csv
```

Example: `rig_log_20260218_162154.csv` → session started February 18, 2026 at 16:21:54.

---

## CSV Column Reference

| Column | Description |
|--------|-------------|
| `t` | Elapsed time since controller start (seconds) |
| `armed` | Arm state (`1` = armed, `0` = disarmed) |
| `thr_us` | Throttle command sent to motors (µs, 1000–2000) |
| `roll_sp_dps` | Roll rate setpoint from RC stick (deg/s) |
| `pitch_sp_dps` | Pitch rate setpoint from RC stick (deg/s) |
| `yaw_sp_dps` | Yaw rate setpoint from RC stick (deg/s) |
| `gx_dps` | Measured gyro roll rate (deg/s) |
| `gy_dps` | Measured gyro pitch rate (deg/s) |
| `gz_dps` | Measured gyro yaw rate (deg/s) |
| `roll_u` | Roll PID total output |
| `roll_p / roll_i / roll_d` | Roll PID proportional / integral / derivative terms |
| `pitch_u` | Pitch PID total output |
| `pitch_p / pitch_i / pitch_d` | Pitch PID P / I / D terms |
| `yaw_u` | Yaw PID total output |
| `yaw_p / yaw_i / yaw_d` | Yaw PID P / I / D terms |
| `m1_us – m4_us` | PWM output to each motor (µs) |
| `rc_raw_roll` | Raw SBUS roll channel value |
| `rc_raw_pitch` | Raw SBUS pitch channel value |
| `rc_raw_thr` | Raw SBUS throttle channel value |
| `rc_raw_yaw` | Raw SBUS yaw channel value |
| `rc_raw_arm` | Raw SBUS arm switch channel value |

---

## Notes

- Log files are flushed to disk every 20 rows (`LOG_FLUSH_EVERY_N = 20`) to limit I/O overhead.
- Add `logs/*.csv` to `.gitignore` to avoid committing large data files. Only commit intentional test result snapshots.
- For nozzle thrust data, see [`../nozzle_test_results/`](../nozzle_test_results/).
