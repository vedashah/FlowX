# FlowX – Space Drone Reconnaissance Platform  
### Cold-gas thruster-based lunar exploration drone (TAMU ESET 429 Capstone)

FlowX is a proof-of-concept **cold-gas propelled reconnaissance drone** designed for low-gravity lunar environments.  
The system is engineered to support future **Permanently Shadowed Region (PSR)** exploration missions by providing high-mobility, low-risk flight capabilities using **CO₂/N₂ cold-gas thrusters**.

---

## Repository Contents

This repository contains:

- All **firmware, control scripts, and drivers** for the FlowX test bench  
- **CAD-linked configuration files**  
- **Electrical schematics** for thrust stand + valves  
- **Data capture scripts** for cold-gas thrust characterization  
- **Documentation** for team workflows, testing procedures, and development environment setup  

---

## Project Overview

FlowX is developed as a two-semester senior capstone project with goals to:

- Create a functional **cold-gas thruster test platform**  
- Measure **impulse, thrust curves, and valve response times**  
- Develop an onboard **embedded control system** using Raspberry Pi + custom MOSFET boards  
- Demonstrate **controlled maneuvering operations** for lunar-like conditions  
- Provide a modular foundation for future **PSR mapping missions**
